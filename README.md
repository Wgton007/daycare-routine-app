Daycare Routine App

Aplicativo web simples para auxiliar na organização da rotina de uma creche comunitária, desenvolvido como projeto extensionista do curso de Análise e Desenvolvimento de Sistemas (ADS – 1º semestre).

O sistema foi pensado para ajudar no cadastro de crianças, controle de presença, geração de relatórios e organização de informações básicas do dia a dia, reduzindo o uso de papel e planilhas manuais.

Objetivo do Projeto:

Este projeto tem como foco social apoiar uma creche comunitária na organização de suas rotinas diárias, facilitando:

Cadastro de crianças e responsáveis

Registro de presença das criancas (check-in)

Geração de relatórios de frequência das criancas

Organização de informações importantes

Criação de backups dos dados


Tecnologias Utilizadas:

HTML5 – Estrutura das páginas

CSS3 – Estilização e layout visual

JavaScript – Lógica do sistema e manipulação dos dados

LocalStorage (Navegador) – Armazenamento local dos dados no computador do usuário

GitHub Pages – Publicação dos Codigos / versão online do sistema

Como Usar o Sistema:

- O sistema funcionará normalmente sem internet.

Funcionalidades:

* Tela de login

* Cadastro de crianças

* Lista de crianças cadastradas

* Registro de presença (Check-in)

* Relatório de presença por data ou intervalo

* Backup dos dados em arquivo JSON

* Impressão ou salvamento de relatórios em PDF

* Armazenamento de Dados

Os dados são salvos localmente no navegador por meio do LocalStorage.
Isso significa que:

Os dados ficam no computador onde o sistema está sendo usado

Para segurança, é possível gerar backups em arquivo .json

Em outro computador, os dados podem ser restaurados importando esse arquivo

Como Testar:

Faça o login no sistema

Cadastre algumas crianças

Acesse a página de Check-in e registre presença

Vá até o relatório e gere um PDF

Teste o backup exportando e importando os dados
